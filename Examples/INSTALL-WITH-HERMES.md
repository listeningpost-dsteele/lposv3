# Example Installation (v3.3)

1. Download `Build/LPOS-Hermes-Compact-v3.3.zip`.
2. Extract it.
3. Start a new Hermes session.
4. Attach the extracted folder or ZIP.
5. Tell Hermes:

```text
Open and execute INSTALL-LPOS-COMPACT.md.

Install LPOS using the compiled distribution. Back up my current environment
first. Preserve existing knowledge, integrations, approved designs, and working
operations.

Load only CHIP-KERNEL.md, CRAFT-STANDARD-ROUTING.yaml, and SPECIALIST-INDEX.md
into context; retrieve everything else by section on demand.

Then run the first-run onboarding in ONBOARDING.md: ask me what to name my
executive office (default Chip), walk me through email setup, and verify the
email loop with a real round trip before activating it.

Do not stop after analysis. Verify the installation before reporting completion.
Do not deploy, send, delete, purchase, or perform an irreversible external
action without my approval.
```

6. Answer the onboarding questions (office name, email address, verified reply
   addresses, available channels).
7. Reply "confirmed" to the test email when it arrives. The 10-minute feedback
   loop activates only after that round trip verifies.
