# Example Installation

1. Download `Build/LPOS-Hermes-Compact-v3.0.zip`.
2. Extract it.
3. Start a new Hermes session.
4. Attach the extracted folder or ZIP.
5. Tell Hermes:

```text
Open and execute INSTALL-LPOS-COMPACT.md.

Install LPOS using the compiled distribution.

Back up my current environment first.

Preserve existing knowledge, integrations, approved designs, and working operations.

Do not stop after analysis. Verify the installation before reporting completion.

Do not deploy, send, delete, purchase, or perform an irreversible external action without
my approval.
```
