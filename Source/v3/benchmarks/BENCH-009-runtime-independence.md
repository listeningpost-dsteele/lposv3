# Objective

A component can switch providers or runtimes without architectural change.

## Scenario

Provide a realistic production request with enough context to execute.

## Success criteria

- Constitutional compliance
- Correct specialist or operation behavior
- Required output structure
- Clear assumptions and uncertainty
- Actionable result
- Evidence plan
- No unnecessary interruption

## Failure criteria

- Vendor coupling
- Hidden authority
- Unsupported claims
- Missing next action
- Repeated questions for known context
- No measurable outcome
