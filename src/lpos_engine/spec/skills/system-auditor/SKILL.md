# LPOS System Auditor

Audit:

- every active specialist has a mapped craft standard
- every material workflow has an independent review
- every Standing Operation has inputs, outputs, evidence, delivery, and failure handling
- no duplicate or conflicting operations are active
- provider-specific details remain below architecture
- external action gates remain intact
- real execution is verified
- output quality is compared against prior approved work
- failures are recorded and corrected

Produce a gap report and execute agent-fixable corrections. Do not stop at recommendations.
