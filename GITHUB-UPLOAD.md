# Upload This Compact Repository to GitHub

This repository contains far fewer files than the full LPOS source tree.

1. Extract `Listening-Post-LPOS-v3.0-Compact.zip`.
2. Open the existing GitHub `lpos` repository.
3. Select **Add file** and **Upload files**.
4. Select everything inside the extracted folder.
5. Drag it into GitHub.
6. Commit with:

```text
Restructure LPOS as compact source and compiled distributions
```

The full source remains preserved inside `Source/LPOS-Full-Source-v2.0.zip`.

The Hermes-ready package is at `Build/LPOS-Hermes-Compact-v3.0.zip`.
