# Listening Post Operating System v3 Compact Repository

This repository is optimized for GitHub, distribution, and Hermes installation.

## Structure

```text
Source/
  LPOS-Full-Source-v2.0.zip

Build/
  Hermes/
  LPOS-Hermes-Compact-v3.0.zip

Tests/
Examples/
```

The complete source is preserved as one archive.

The Hermes build is compiled into a much smaller install package.

## Install

Use:

`Build/LPOS-Hermes-Compact-v3.0.zip`

## Develop

Extract:

`Source/LPOS-Full-Source-v2.0.zip`

Make changes in the full source repository, then rebuild the compact distribution.
